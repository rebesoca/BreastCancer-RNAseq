# RNA-seq Nextflow pipeline

Pipeline basado en el workflow implementado en Galaxy.

## Estructura
- main.nf → pipeline principal
- bin/script_deseq2.R → análisis diferencial
- data/ → FASTQ (no incluidos)
- genome/ → índice STAR (no incluido)
- annotation/ → GTF (no incluido)

## Nota
El pipeline no se ejecuta debido a que los datos originales se procesaron en Galaxy.
