## RNA-seq Differential Expression Analysis of Triple-Negative Breast Cancer Organoids

This repository contains the bioinformatics pipeline and analysis scripts developed for the differential expression analysis of RNA-seq data from patient-derived triple-negative breast cancer (TNBC) (GEO accession: GSE287429).
The dataset includes 12 samples from three experimental conditions — untreated control (CTR), doxorubicin (DOX) and bortezomib (BTZ) — evaluated at two time points (23 h and 47 h) with two biological replicates per condition. Libraries were prepared using the TruSeq Stranded mRNA protocol and sequenced on an Illumina NovaSeq X Plus platform.
